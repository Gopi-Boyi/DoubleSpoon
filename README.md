# 🍽️ DoubleSpoon — Food Ordering Website

> *"Good food choices are good investments."*

A clean, responsive frontend food ordering website built with HTML, CSS, and JavaScript.

🌐 **Live Demo:** [gopi-boyi.github.io/DoubleSpoon](https://gopi-boyi.github.io/DoubleSpoon/)

---

## 📸 About the Project

**DoubleSpoon** is a food-themed frontend website that showcases a modern restaurant/food ordering experience. It highlights healthy, handcrafted meals made from the finest ingredients — designed to make visitors feel hungry and engaged!

---

## ✨ Features

- 🏠 **Hero Section** — Catchy headline with a strong call-to-action
- 🥗 **Explore Foods** — Menu section with food items, prices, and order buttons
- 👨‍🍳 **About Section** — Story behind the brand and cooking philosophy
- ⭐ **Testimonials** — Customer reviews carousel
- ❓ **FAQ Section** — Frequently asked questions accordion
- 📧 **Newsletter Subscription** — 25% OFF offer for subscribers
- 📱 **Responsive Design** — Works across desktop and mobile devices

---

## 🛠️ Tech Stack

> ⚠️ This is a **Frontend Only** project — no backend or database involved.

| Technology | Usage |
|------------|-------|

## 🚀 Getting Started

### View Live
👉 [https://gopi-boyi.github.io/DoubleSpoon/](https://gopi-boyi.github.io/DoubleSpoon/)

### Run Locally

```bash
# Clone the repository
git clone https://github.com/Gopi-Boyi/DoubleSpoon.git

# Navigate into the folder
cd DoubleSpoon

# Open in browser
open index.html
```
Or simply drag and drop `index.html` into your browser.
